
### Requirements 
To compile this program i recommend using Visual Studios 2019 with the following compiler settings
* Set the linker subsystem value to /SUBSYSTEM:windows. This can be found under project Properties->Linker->System->Subsystem.
* Set the entry point to mainCRTStartup. This can be found under project Properties->Linker->Advanced->Entry Point.
* Set the runtime to Multi-threaded(/MT). This can be found under project Properties->C/C++->Code Generation->Runtime Library.

This program has only been tested on Windows 10 OS.
